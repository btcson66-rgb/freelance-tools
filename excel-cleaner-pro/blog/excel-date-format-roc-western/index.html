手動做法：先用看得懂的流程處理

保留原日期欄，分類民國年與西元年，民國年加 1911 後再組成真正日期。手動流程雖然慢，但很適合第一次建立規則，因為每一步你都知道自己為什麼這樣做。

  1. 複製原始檔，檔名加上日期與未清理字樣。
  2. 確認每一欄只放一種資料，不要把備註和主要欄位混在一起。
  3. 開啟篩選，逐欄找空白、異常值、格式不同與看起來重複的資料。
  4. 先標示疑似問題，不要一開始就刪除整列。
  5. 把規則寫在清理紀錄欄，例如判斷欄位、保留條件與人工確認原因。
  6. 處理後記錄原始筆數、刪除筆數、待確認筆數與完成筆數。
  7. 抽樣檢查前段、中段、後段資料，再另存清理後版本。

常見錯誤與檢查重點

只改儲存格顯示格式，無法把文字日期變成可排序日期。另一個常見錯誤，是只修畫面上看得到的列，卻忘記取消篩選後還有其他資料。清理完也要重新檢查總筆數、總金額或關鍵欄位，不要只相信畫面看起來乾淨。

  • 直接在原始檔操作，事後無法追查。
  • 沒有先定義唯一欄位，導致誤刪或漏刪。
  • 把電話、編號、身分證字號當成數字，造成前導 0 消失。
  • 用全表尋找取代，誤改地址、備註或品號。
