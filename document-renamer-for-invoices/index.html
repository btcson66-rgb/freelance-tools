Automatically rename invoices and receipts without uploading them

FilePilot Local is a Windows app that reads invoices, receipts, scans and PDFs locally, extracts the filing details, then renames files into a consistent pattern.

The problem: filenames do not match the document

Invoices and receipts often arrive as scan.pdf, receipt.jpg, download (4).pdf or a vendor-specific name that is not useful later. Renaming by hand is slow, and uploading private financial documents to an online renamer may not be acceptable.

How automatic invoice renaming works

  1. Read the document locally.
    FilePilot Local extracts text from PDFs and uses OCR for scanned PDFs and photos.
  2. Find the useful fields.
    It looks for date, company, amount and document type.
  3. Apply a filename template.
    Use a pattern such as {date}_{company}_{documentType}_{amount}.
  4. File by document type.
    Invoices go to Invoices, receipts to Receipts, quotes to Quotes, and bank files to Bank.
  5. Separate uncertain files.
    Low-confidence files go to Needs Review. Duplicates are flagged to a Duplicates folder.

Offline vs cloud invoice renaming

QuestionFilePilot Local
Are documents uploaded?No. Processing happens on your Windows computer.
Does it use cloud AI?No cloud AI API is used.
What if a scan is unclear?It goes to Needs Review instead of being silently filed as certain.
What if there are duplicates?They are flagged to a Duplicates folder, not auto-deleted.
Can the same naming pattern be reused?Yes. Templates use extracted fields such as date, company, document type and amount.

When this is a good fit

FilePilot Local is useful when you have a recurring pile of invoices and receipts, need searchable filenames, and do not want those files uploaded. It is especially relevant for Taiwan folders that mix English documents with ROC dates, NT$ amounts, tax IDs and invoice numbers.

It is not meant to replace judgment for unclear documents. The Needs Review folder is there because automatic extraction should be visible when confidence is low.
