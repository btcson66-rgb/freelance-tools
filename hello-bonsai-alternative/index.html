Frequently asked questions

Is ProposalPro really free?
The core tool is free to use with no account. Pro adds extra features as a one-time purchase, not a subscription.
Does it work offline like Hello Bonsai doesn't?
Yes. ProposalPro runs entirely on your device with no internet required, whereas Hello Bonsai is a cloud service.
Can it replace Hello Bonsai for a team?
Not really. ProposalPro is built for solo freelancers. If you need team collaboration and a full platform, Hello Bonsai is the better fit.
Where is my data stored?
On your own machine, in a file you own. It never goes to a vendor's servers.
Do I lose my data if I stop paying?
No. Because the file is local and yours, there is no subscription that can lock you out of your own records.
