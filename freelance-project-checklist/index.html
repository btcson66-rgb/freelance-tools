The freelance project steps most people skip

Every freelance project goes through the same phases — and every freelancer has their own version of "I forgot to ask for the deposit" or "I sent the final invoice but never asked for a review." A checklist is the simplest possible fix: the same steps, applied consistently, for every project.

This tool has all the standard steps pre-loaded across four phases. Create a new project with one click, work through the list as you go, and tick off items as they are done. You can add your own steps, reorder them, and archive completed projects so the active list stays clean.

The 4-phase freelance project checklist

Phase 1: Before you start

  • Quote or estimate sent
  • Contract signed by both parties
  • Deposit received
  • Kickoff call or meeting done
  • Brief confirmed in writing

Phase 2: During the work

  • Milestones reviewed with client
  • Change requests documented and priced
  • Time logged (if billing hourly)

Phase 3: Delivery

  • Final files delivered
  • Client sign-off received
  • Final invoice sent
  • Late fee clause ready if payment is delayed

Phase 4: After the project

  • Testimonial or review requested
  • Case study or portfolio piece added
  • Referral conversation had
  • Next project discussed
