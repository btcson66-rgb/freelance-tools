統一 Excel 裡的電話號碼格式

Excel Cleaner Pro 部落格 · 2026-06-09

電話號碼是名單裡最容易亂的欄位：開頭的 0 不見了、市話手機混排、有人加區碼有人沒加。格式不一致，撥號、比對、匯入系統都會出問題。

為什麼開頭的 0 會消失

最常見的困擾是手機號碼 0912345678 變成 912345678。原因是 Excel 把它當成「數字」，而數字開頭的 0 沒有意義就被去掉了。解法是把電話欄位先設成「文字」格式再貼上，或在輸入時讓整欄維持文字屬性，0 就會被保留下來。電話號碼本質上是代號不是數量，本來就該用文字存。

先決定一個統一格式

整理前先定義目標格式：手機要不要加區隔（0912-345-678 還是 0912345678）、市話要不要含區碼（02-1234-5678）。把規則訂清楚，整份名單照同一套走，後續無論是匯入系統還是人工撥號都會順很多。沒有絕對正確的格式，重點是「一致」。

用公式做基本清理

如果只是要去掉多餘的空白、括號或分隔符號，可以用 SUBSTITUTE 把不要的字元換掉，例如 =SUBSTITUTE(SUBSTITUTE(A2," ",""),"-","") 先清成純數字串，再依需要重新加上分隔。市話補區碼則比較麻煩，因為要判斷號碼長度，公式會變長，資料不一致時容易出錯。

大量名單用批次工具

當名單動輒上千筆、格式又五花八門時，用離線的 Excel 整理工具把整欄電話一次正規化（保留開頭 0、統一分隔、清掉雜訊）會比逐格修公式快很多，而且資料都在自己電腦上處理，客戶電話不會外流。這只是其中一種做法，重點是先把規則定清楚。

市話與手機的判斷

台灣手機是 10 碼、09 開頭，市話則含區碼長度不一。批次整理時若要區分兩者，可依長度和開頭判斷，但遇到分機、國碼 +886、舊式號碼就要特別處理。建議先把明顯規則的處理掉，剩下少數特例再人工檢查，不必硬要一條規則涵蓋所有狀況。

整理完的檢查重點

清完後抽查幾筆，確認開頭 0 都還在、號碼碼數合理（手機 10 碼）、沒有把分機號或國碼誤刪。電話資料牽涉聯絡得上聯絡不上，寧可整理後多花幾分鐘核對，也不要等到撥不通才發現號碼被改壞了。

常見問題

電話號碼開頭的 0 為什麼會不見？

因為 Excel 把它當數字處理，開頭 0 被去掉。把欄位先設成文字格式再輸入或貼上，就能保留開頭 0。

怎麼把電話統一成同一種格式？

先決定目標格式（要不要分隔、市話要不要含區碼），再用 SUBSTITUTE 清除雜訊或用批次工具一次正規化整欄。

市話要補區碼怎麼做？

要依號碼長度判斷，公式會較複雜且資料不一致時易錯。建議先處理規則明確的，少數特例再人工補。

處理大量電話會外洩嗎？

用離線工具或本機 Excel 處理，資料都在自己電腦上，不會上傳，適合含客戶聯絡資訊的名單。

整理完要檢查什麼？

抽查開頭 0 是否保留、碼數是否合理、有沒有誤刪分機或國碼，確認撥得通再交付使用。
