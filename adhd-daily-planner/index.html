FocusFlow is built around ADHD-friendly planning principles: less blank-page pressure, fewer choices at once, visual time support, and forgiving controls for skipping, postponing, and restarting.

  • Executive dysfunction: tasks can start smaller, with gentle defaults already in place.
  • Task paralysis: brain dump sorting reduces the number of decisions on screen.
  • Time blindness: color blocks and the 1.5 estimator make time more visible.
  • Body doubling: the language and structure are designed to feel like quiet support beside you.

FAQ

Is FocusFlow free?

FocusFlow Lite is free. FocusFlow Pro is $15 and adds routines, week/month views, custom themes, custom quotes, and fuller planning controls.

Does it work offline?

Yes. The planner is a single HTML file with inline CSS and JavaScript. It stores data in your browser using localStorage.

Why multiply time by 1.5?

It adds cushion for starting, switching, transitions, and the common ADHD pattern of underestimating how long a task will take.

Does FocusFlow use streaks?

No. The planner avoids streak mechanics because broken streaks can create shame and make restarting harder.

FocusFlow is a self-contained offline planner. It is not medical advice and does not diagnose ADHD.
