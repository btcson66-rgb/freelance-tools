基本介紹

Excel Cleaner Pro 是什麼？

Excel Cleaner Pro 是一個 Windows 桌面軟體，專門用來整理髒亂的 Excel/CSV 資料。不需要會寫函數，只要把檔案拖進來、選要做的操作、按幾個按鈕就能完成去重複、格式統一、名單比對等清理工作。所有資料處理都在你自己的電腦上完成，不會上傳到雲端。

適合誰用？

辦公室行政人員、HR、業務助理、會計、中小企業主——任何「每天要整理 Excel 名單、但不會寫函數」的人。如果你已經很熟 Power Query 或 Python/pandas，這個工具對你幫助有限。

支援什麼檔案格式？

支援 .xlsx（Excel 2007+）、.xls（舊版 Excel 97-2003）、.csv（逗號/Tab/分號分隔都可以）。處理結果另存新檔（.xlsx），不會覆蓋原始檔。

系統需求是什麼？

Windows 10 或 Windows 11（64 位元），約 100MB 硬碟空間。不需要安裝 Python 或其他軟體。不支援 macOS 或 Linux。

功能相關

有哪些功能？

10 個主要功能：(1) 刪除重複資料 (2) 批次格式統一（日期、電話、身分證）(3) 欄位拆分 (4) 欄位合併 (5) 批次資料夾處理 (6) 名單比對 (7) 缺失值檢查 (8) 異常資料偵測 (9) 一鍵輸出整理後 Excel (10) 產生整理報告。

民國日期轉換怎麼運作？

自動辨識各種民國年寫法（113/01/05、民國113年1月5日、113.01.05 等），統一轉換成你指定的格式。也可以反向從西元轉民國。不需要手動指定來源格式——程式自動判斷。

身分證驗證做什麼？

自動轉大寫 → 用台灣身分證加權檢查碼公式驗算 → 標出不合法的號碼。適合 HR、客服、金融業整理名單。

名單比對怎麼用？

拖入兩個檔案（或兩個工作表），選比對欄位（email、員工編號等），程式告訴你：兩邊都有（交集）、只在 A 不在 B、只在 B 不在 A。結果匯出成新 Excel。

批次資料夾處理是什麼？

選一個資料夾，裡面所有 Excel/CSV 套用相同清理規則，一次處理完。每個檔案各生一份整理版，外加合併報告。適合月報、門市報表等重複性工作。

異常偵測用的是 AI 嗎？

不是。完全是規則式：離群值（IQR倍數）��型態不符（正規表示式）、不可能的日期（邏輯判斷）、重複ID、前後空白。同樣輸入永遠同樣結果。

整理報告長什麼樣子？

兩種格式：HTML（瀏覽器直接看，可寄給同事）和 XLSX（Excel 打開做後續分析）。內容：做了哪些操作、影響幾筆、前後差異、異常清單。

可以處理多大的檔案？

取決於電腦記憶體。10 萬列以內通常都沒問題。建議 8GB 以上記憶體。

會不會覆蓋我的原始檔？

不會。永遠另存新檔（預設加 _cleaned 後綴）。原始檔完全不動。

疑難排解

啟動時出現錯誤怎麼辦？

常見原因：(1) Windows 版本太舊（需要 Win10 以上 64位元）(2) 防毒軟體誤判（加入白名單即可）(3) 路徑有特殊字元。如果問題持續，請截圖錯誤訊息寄到 btcson66@gmail.com。

授權碼輸入後顯示失敗？

確認：(1) 授權碼有正確複製（沒有多餘空格）(2) 電腦有連上網路 (3) 防火牆沒有擋 api.gumroad.com。如果確認以上都沒問題仍失敗，請聯繫支援。

處理很慢怎麼辦？
